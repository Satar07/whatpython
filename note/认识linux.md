# 认识linux

## 目录

1. 查看目录中的内容
- ```ls``` 蓝色文件夹 白色文件
2. 进入目录 
``` cd ``` 
- 快捷键```..```返回上一级 
 ```-```回退到上一次的目录 
 ```~```返回主目录
3. 获取绝对路径 ```pwd ``` 
- 可以使用tab键补全文件名 按两下显示全部文夹
4. 创建目录 ```mkdir``` 
- ```-p``` 快速创建多级目录
>``` mkdir -p one/two/three```
5. 显示文件夹树状图 ``` tree (文件目录)```
##文件
1. 创建空白文件 ```touch 文件目录``` 
- 该命令不会覆盖原有文件
2. 复制 ```cp 文件目录 目标目录``` 
- ```-r ```复制文件夹
3. 删除 ```rm 文件目录```
- ```-r```删除的是文件夹
4. 移动 ```mv 文件目录 目标目录```
5. 重命名 ```mv 文件目录 新名字```
6. 查看文件内容 ```cat``` 
- ```-n```可以带行号的查看
##指令
1. 查看帮助``` 指令 --help```

>学习最好途径是利用搜索引擎并亲自在linux上码出来：）
